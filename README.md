# SudokuSolver
 
A simple sudoku solver in Python using backtracking and forward-checking algorithm. Prints result and number of backtracking steps at the end.

## Usage:
```
usage: sudoku.py [-h] [-fc] sudoku

Sudoku solver using backtracking and forward-checking algorithms

positional arguments:
  sudoku      a line of text representing sudoku board, 81 characters with . representing an empty square,
              example: ......2.6...7....8.5......36.1.5.32.....37.....2......9...7......36..4915..81....

optional arguments:
  -h, --help  show this help message and exit
  -fc, --fc   enables forward-checking
```
